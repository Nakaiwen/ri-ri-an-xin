圖片載入順序（fallback chain）
═══════════════════════════════════════════════════════

App 載入訪客圖時會依序嘗試：

1. 「動物名_day.jpg」或「動物名_night.jpg」← 優先
2. 「動物名.jpg」← 通用版（單一圖片）
3. 對應的 emoji（🐢 🦋 🦉 …）← 最後 fallback

所以你可以分階段準備：
- 完全沒做：顯示 emoji
- 只放單張通用版（例如 kiwi.jpg）：所有時段都用它
- 完整準備白天/晚上兩版：依時段自動切換
